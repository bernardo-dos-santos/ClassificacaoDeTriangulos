# Classificação de Triângulos

Um programa muito útil para verificação se tal Triângulo é possível e qual seu respectivo tipo

# Funcionalidades

**Validação de Triângulo** - Verifica se o triângulo pode ser existente

**Verificação de Qual Triângulo** - Identifica Qual tipo de triângulo é entre: Escaleno, Isósceles e Equilátero

**Validação de Entrada** - Confere se o usuário digitou números válidos

# Requisitos 

.NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.

# Como Utilizar

1. Clone o Repositório
```
git clone https://github.com/bernardo-dos-santos/ClassificacaoDeTriangulos
```
2. Navegue até a pasta raiz da solução
```
cd ClassificacaoDeTriangulos
```
3. Restaure as dependências
```
dotnet restore
```
4. Navegue até a pasta do projeto
```
cd ClassificacaoDeTriangulos.ConsoleApp
```
5. Execute o projeto
```
dotnet run
```
